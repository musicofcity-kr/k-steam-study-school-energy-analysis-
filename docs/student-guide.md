# 학생용 사용 안내

## 오늘의 미션

학교의 전기가 어디에서 오고 어떤 과정을 거치는지 알아본 뒤, 전력 사용 데이터를 근거로 2050 미래학교 에너지 시스템을 설계합니다.

이 앱은 실제 학교나 도시를 설계하는 전문 도구가 아니라 중1 수업용 비교 도구입니다.

## 1. 전기는 어디서 올까?

발전소에서 만든 전기는 송전선, 변전소, 지역 전력망을 거쳐 학교로 들어옵니다.

- 화력: 연료의 열로 물을 끓여 증기·터빈·발전기를 움직임
- 원자력: 핵분열의 열로 물을 끓여 증기·터빈·발전기를 움직임
- 수력: 흐르는 물로 터빈과 발전기를 움직임
- 풍력: 바람으로 날개와 발전기를 움직임
- 태양광: 태양전지가 빛을 전기로 직접 바꿈

## 2. 왜 발전소는 학교 옆에 없을까?

화면에 보이는 필수 기술 카드 4개를 모두 확인하고 다음 위치 중 하나로 분류합니다. 아래 O/X 4문항까지 마치면 미션 2가 완료됩니다.

- 학교 안에 설치 가능한 기술
- 조건이 맞을 때 가능한 지역 기술
- 학교 밖 발전소에서 전력망을 통해 공급하는 기술

“위험해서 안 된다”만 쓰지 말고 공간, 자연조건, 안전시설, 연료 운송, 소음, 비용도 함께 생각합니다. 원자력은 학교 안에 배치하지 않습니다. 수소 연료전지는 공급·저장 안전·공간·비용·생산 방식이 확인된 조건부 기술입니다.

## 3. 전력 패턴 탐정

`수업 데이터로 시작하기`를 누르거나 선생님이 준 CSV를 엽니다.

- `수업용 가정 데이터`: 실제 학교 전력량이 아님
- `교사가 준비한 공공데이터`: 제공기관과 자료명을 확인
- `출처 확인 필요`: 파일명만으로 출처를 알 수 없음

그래프에서 피크·최저 시간, 태양광 활동 시간을 비교하고 피크 원인을 추론합니다. 그래프만으로 원인을 확정할 수 없으면 추가 조사가 필요하다고 적습니다.

## 4. 미래학교 시스템 설계

조절 항목은 다음 다섯 가지입니다.

1. 학교 지붕·주차장 태양광
2. ESS 저장 용량
3. 조건부 수소 연료전지
4. 에너지 절감
5. 스마트 에너지 관리

원자력 슬라이더는 없습니다. 원자력은 학교 밖 전력망 공급원 토론 카드에서만 다룹니다.

태양광은 설정된 낮 시간에만 발전합니다. 남은 태양광은 ESS에 저장되고, 이후 부족한 시간에 방전됩니다. 그래도 부족한 전기는 외부 전력망이 자동으로 공급합니다. 외부 전력망을 쓰는 것은 실패가 아닙니다.

설계안 A와 B를 저장해 지역 에너지 충당률, 외부 전력망 사용량, 피크 대응 정도와 장소 조건을 비교한 뒤 최종안을 선택합니다.

## 5. 보고서와 발표

보고서에는 다음을 포함합니다.

- 사용한 데이터와 출처
- 피크 시간과 추론 근거
- 학교 안에 배치한 기술
- 조건부로 활용한 지역 기술
- 학교 밖 전력망을 통해 이용한 기술
- 해당 위치로 구분한 이유
- 설계의 장점과 한계
- 실제 적용 전에 필요한 자료
- 1분 발표문

수업용 데이터를 사용한 보고서에는 실제 학교 전력량이 아니라는 경고문이 자동으로 들어갑니다.

- 모둠 이름에는 실명·학번 대신 별칭을 씁니다.
- 보고서에도 이름, 학번, 연락처를 적지 않습니다.
- 학생 화면에서는 복사와 인쇄로 결과를 제출합니다.

## 저장과 다시 시작

- 활동은 현재 브라우저 탭에 자동 저장됩니다.
- 새로고침하면 `저장한 미션 이어하기`로 계속할 수 있습니다.
- 다른 모둠의 기록이 보이면 상단 `처음부터`를 누르고 확인합니다.
- `처음부터`는 미션 답, 데이터, 설계안과 보고서를 모두 지우므로 확인 문구를 읽고 선택합니다.

## 꼭 기억할 점

- ESS는 전기를 만들지 않고 저장합니다.
- 절감과 스마트 관리는 발전량이 아닙니다.
- 태양광은 밤에 발전하지 않습니다.
- 수소는 어떻게 만들었는지에 따라 환경성이 달라집니다.
- 지역 충당률이 높다고 무조건 좋은 설계는 아닙니다.
- 실제 적용에는 일사량, 설치 면적, 효율, 비용, 안전, 법, 전력망 조건을 더 조사해야 합니다.
