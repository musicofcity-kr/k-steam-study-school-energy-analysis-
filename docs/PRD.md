# E-CITY 2050 Energy Design Lab PRD

## 목적

중학교 1학년 학생이 전력 사용 CSV 데이터를 읽고, 에너지원의 역할과 한계를 비교한 뒤, 2050년 탄소중립 미래 학교구역 설계안을 근거 있게 설명하도록 돕는 수업용 STEAM 웹앱을 만든다.

## 범위

- 단일 페이지 React/Vite/TypeScript 정적 앱
- 브라우저 CSV 업로드와 수업 연습용 예시 데이터
- 시간대별 전력 사용량 탐색, 요약 지표, 그래프
- 에너지원 비교 카드 6종
- 태양광, ESS, 수소, 차세대 원자력, 절감률 슬라이더 기반 교육용 시나리오 모델
- 교사용 가정값 설정 패널
- 학생 보고서 초안, 복사, 인쇄, localStorage 저장
- README, 교사용 안내, 학생용 안내

## 비범위

- 실제 도시 에너지 설계를 대체하는 공학 계산
- 외부 공공데이터 API 직접 호출
- 실제 지역 전력량, 발전량, 탄소배출계수의 임의 생성
- 차세대 원자력 설치 권장 표현

## 핵심 사용자

- 학생: 중1 수준, 그래프와 단계 안내가 필요하고 근거 기반 설명을 연습한다.
- 교사: CSV를 준비하고 수업용 가정값을 조정하며 학생 결과물을 활동지나 발표에 활용한다.

## 필수 기능

1. 시작 화면과 활동 단계 안내
2. 예시 데이터 불러오기와 CSV 업로드
3. CSV 컬럼 자동 감지와 수동 매핑
4. 데이터 미리보기 표
5. 시간대별 전력 사용량 그래프와 피크 강조
6. 총 사용량, 평균 사용량, 최대/최소 시간대 계산
7. 에너지원 비교 카드
8. 미래 학교구역 설계 슬라이더
9. ESS를 발전량이 아닌 안정성/피크 대응 점수에 반영
10. 교육용 비교 지표 안내 문구
11. 결과 보고서 자동 초안과 수정 textarea
12. 복사, 인쇄, JSON 저장, localStorage 저장
13. 교사용 가정값 설정과 한계 안내
14. 학생 친화 오류 메시지

## 데이터 원칙

- 샘플은 항상 "수업 연습용 예시 데이터"로 표시한다.
- 실제 공공데이터가 아니며 교사가 CSV로 교체할 수 있음을 화면과 문서에 표시한다.
- 공식값처럼 보이는 기본값을 만들지 않는다. 모든 기본 가정값은 수업용 비교값이다.
- 개원중학교 데이터 수집팩은 `서울특별시 법정동별시간별전력사용량` 중 `개포동` 행을 찾아 쓰도록 안내하되, 팩에 없는 `usage_kWh` 값은 임의 생성하지 않는다.

## 오류 처리

- 빈 CSV
- CSV가 아닌 파일
- 시간 열 또는 전력사용량 열 미감지
- 숫자가 아닌 시간/사용량
- 음수 전력사용량
- 일부 잘못된 행 제외와 제외 행 수 안내

## 성공 기준

- `npm install`, `npm test`, `npm run build`가 실행 가능하다.
- TypeScript 빌드 오류가 없다.
- 학생이 첫 화면에서 10초 안에 시작할 수 있다.
- 학생이 보고서에 "공공데이터에서 찾은 전력 사용 패턴을 근거로 미래 학교구역 에너지 조합을 설계했다"는 문장을 완성할 수 있다.
