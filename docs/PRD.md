# E-CITY 2050 수업 적용 PRD v2

## 목적

중학교 1학년 학생이 현재의 발전방식과 전력 전달 과정을 이해하고, 발전·저장·절감·관리·외부 공급을 구분한 뒤, 전력 사용 데이터를 근거로 2050 미래학교 에너지 시스템을 설계하고 한계까지 설명하도록 돕는다.

## 핵심 학습 성과

학생은 활동 후 다음을 자신의 말로 구분한다.

1. 화력·원자력·수력·풍력은 터빈과 발전기를 이용하고 태양광은 빛을 직접 전기로 바꾼다.
2. 대규모 발전시설은 공간, 자연조건, 안전시설, 연료 운송, 소음, 비용 등 때문에 학교 주변에 설치하기 어렵다.
3. ESS는 발전원이 아니라 저장 장치이고, 절감과 스마트 관리는 전기를 새로 만들지 않는다.
4. 원자력은 학교 안 시설이 아니라 학교 밖 발전소에서 전력망을 통해 공급되는 전원의 한 종류다.
5. 수소 연료전지는 공급·안전·공간·비용·생산 방식이 확인된 조건부 지역 보조 기술이다.
6. 미래학교는 태양광, ESS, 절감, 스마트 관리와 외부 전력망을 하나의 시스템으로 연결한다.

## 사용자 흐름

1. `전기는 어디서?`: 발전소에서 학교까지의 흐름과 발전 원리 확인, 2문항 개념 확인
2. `학교 주변 적합성`: 6개 발전 기술 중 4개 이상 확인, 학교 안·조건부 지역·외부 전력망 분류
3. `전력 패턴 탐정`: 수업용 데이터 또는 CSV 로드, 피크·최저 확인, 피크 원인 추론
4. `미래학교 시스템`: 태양광·ESS·조건부 수소·절감·스마트 관리 조절, 설계안 A/B 비교 후 최종안 선택
5. `보고서 & 발표`: 데이터 출처, 장소 적합성, 외부 공급, 장점·한계·추가 조사와 1분 발표문 작성

## 필수 기능

- 발전 원리와 전력 전달 여정
- 기술 10종의 역할·입지·조건 카탈로그
- 즉시 해설이 있는 개념 확인
- 기존 CSV 자동 감지·수동 매핑·CP949/EUC-KR 디코딩 유지
- 데이터 출처 상태와 공공데이터 상세 기록
- 낮 시간대에만 태양광 발전
- 시간별 ESS 충전·방전과 외부 전력망 부족분 계산
- 지역 에너지 충당률, 외부 전력망 사용량, 피크 대응 정도 우선 표시
- 설계안 A/B 저장·비교·최종 선택
- 학생 모드에서 교사용 가정값 숨김
- 설계 초기화와 전체 수업 초기화
- 장소 적합성과 한계를 포함한 보고서·JSON·인쇄·브라우저 저장
- 저장 형식 version 2와 구버전 안전 마이그레이션

## 데이터 원칙

- 수업용 24시간 데이터는 총 900 kWh, 피크 14시 67 kWh를 유지한다.
- 수업용 데이터는 항상 `수업용 가정 데이터`로 표시한다.
- 업로드 파일명만으로 공공데이터라고 판정하지 않는다.
- 공공데이터에는 제공기관, 자료명, 기준일, 지역 단위, 학교 자체/지역 대체 여부를 기록한다.
- 개원중학교 실제 전력량을 생성하거나 추정하지 않는다.
- 모든 교사용 기본값은 실제 설비값이 아닌 수업용 가정값이다.

## 계산 의미

- 지역 에너지 충당률은 절감 후 수요 중 지역 태양광·조건부 수소·ESS 방전으로 충당한 비율이다.
- 외부 전력망 사용량은 부족분이며 지역 충당률에 포함하지 않는다.
- ESS 충전과 방전은 발전량이 아니다.
- 스마트 관리는 피크 수요 또는 ESS 운용을 돕지만 발전량에 합산하지 않는다.
- 수소 환경성은 생산 방식이 `green`일 때만 제한적으로 가산하고, 미확인일 때 가산하지 않는다.

## 비범위

- 실제 학교·도시 설계용 공학 계산
- 실제 개원중학교 전력량 생성
- 원자력 시설의 학교 내부 배치
- 외부 공공데이터 API 직접 호출
- 학생 개인정보 수집 또는 서버 저장
- 점수만으로 최적 설계를 단정하는 기능

## 성공 기준

- 새 5단계 수업 흐름과 완료 조건이 화면에 반영된다.
- 선행 미션 완료 전 다음 미션 이동과 A/B 설계 저장이 잠긴다.
- 연습 데이터는 다른 출처 유형으로 바꿀 수 없고 보고서에서도 수업용 가정 데이터로 유지된다.
- 학생이 수정한 보고서는 명시적 재생성으로 최신 데이터와 설계를 다시 반영할 수 있다.
- 학생 화면은 교사용 출처 편집·원천 URL·컬럼 매핑·JSON 저장을 숨긴다.
- 학생 진행은 현재 탭에 자동 저장되고, 초기화는 확인 대화상자를 거친다.
- 실명·학번·연락처 대신 모둠 별칭을 사용하도록 화면에서 안내한다.
- 태양광 야간 발전량이 0이고 ESS 용량·충방전 한계를 넘지 않는다.
- 외부 전력망 부족분과 지역 충당률 계산이 일치한다.
- 구버전 `nuclearLevel` 저장값을 새 로컬 발전량으로 변환하지 않는다.
- 학생 모드에서 교사용 패널이 보이지 않는다.
- 모바일·태블릿에서 가로 스크롤 없이 핵심 흐름을 완료할 수 있다.
- `npm test`와 `npm run build`가 통과한다.
