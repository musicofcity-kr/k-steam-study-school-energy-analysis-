# E-CITY 2050 시간별 에너지 모델 안내

## 공통 원칙

- 모든 공급량과 점수는 중1 수업용 비교 모델의 결과이며 실제 학교 설비값이 아니다.
- 태양광·수소·ESS·절감·스마트 관리의 기본값은 교사가 수업에서 비교하기 위한 가정값이다.
- 지역 에너지 충당률이 높다고 무조건 좋은 설계로 평가하지 않는다.
- 실제 적용에는 일사량, 설치 면적, 설비 효율, 비용, 법·안전 기준과 전력망 조건이 필요하다.

## 시간별 처리 순서

CSV의 각 행을 입력 순서대로 처리하며 ESS 초기 저장량은 0 kWh다.

1. 전력 수요에 기본 절감률을 적용한다.
2. 원 데이터의 피크 시간에는 스마트 관리 추가 절감 효과를 적용한다.
3. 태양광 활동 시간에만 태양광 발전량을 계산한다.
4. 조건부 수소 연료전지 공급량을 계산한다.
5. 태양광을 먼저 현재 수요에 사용하고 수소가 남은 수요를 보충한다.
6. 남은 태양광만 ESS에 충전한다.
7. 부족한 전력은 ESS에서 방전한다.
8. 그래도 부족한 양을 외부 전력망 사용량으로 기록한다.
9. 충전하지 못한 태양광과 사용하지 못한 수소를 잉여 전력으로 기록한다.

## 태양광

```text
활동 시간의 시간당 태양광 발전량 =
태양광 100% 활동 시간당 공급 가정값 × 태양광 슬라이더 비율
```

설정된 시작·종료 시간은 발전 시간에 포함한다. 활동 시간 밖의 태양광 발전량은 0이다. 실제 태양광은 날씨, 계절, 패널 방향과 음영에 따라 달라지지만 이 모델에서는 활동 시간 안에서 같은 시간당 가정값을 사용한다.

## ESS

ESS 비율은 최대 저장 용량과 시간당 충전·방전 한도를 함께 조절한다.

- 태양광 잉여만 충전한다.
- 저장 용량과 시간당 최대 충전량을 넘지 않는다.
- 현재 저장량과 시간당 최대 방전량보다 많이 방전하지 않는다.
- 왕복 효율 손실은 충전할 때 한 번 적용한다.
- ESS 방전량은 저장했던 전력의 이동이며 새로운 발전량이 아니다.

## 스마트 에너지 관리

스마트 관리 슬라이더는 발전량에 더하지 않는다. 수업용 단순 모델에서는 원 데이터의 피크 시간대 수요를 교사용 최대 추가 절감률 범위에서 조금 더 줄인다.

## 지역 에너지 충당률과 외부 전력망

```text
지역 에너지 충당률 =
(절감 후 소비량 - 외부 전력망 사용량)
/ 절감 후 소비량 × 100
```

지역 충당률은 0~100%로 표시한다. 외부 전력망 사용량이나 학교 밖 원자력 공급은 지역 충당률에 포함하지 않는다. 외부 전력망은 지역에서 부족한 전력을 보충하는 정상적인 연결 시스템이다.

`supplyKWh`는 태양광·수소를 현재 수요에 직접 사용한 양과 ESS가 방전해 실제 수요에 사용한 양의 합이다. 생산했지만 쓰거나 저장하지 못한 전력은 포함하지 않는다.

## 수소 생산 방식

- `green`: 재생전력 기반 생산을 가정하고 환경성 비교 점수에 제한적으로 반영
- `mixed`: 환경성 점수 가산 없음
- `unspecified`: 생산 방식 미확인으로 환경성 점수 가산 없음

수소는 연료전지 사용 단계뿐 아니라 생산 방식, 저장과 운송도 확인해야 한다.

## 상세 점수

다양성·환경성·현실성 점수는 실제 공학 평가가 아니라 설계 토론을 위한 보조 지표이며 학생 화면에서는 `상세 비교 보기` 안에 둔다. 계수는 `src/data/scoreWeights.ts`에 모여 있다.

- 피크 대응: ESS, 절감, 조건부 수소, 스마트 관리, 전략 다양성
- 다양성: 사용한 전략 수와 한 전략 쏠림 정도
- 환경성: 태양광, 절감, ESS, 스마트 관리, 확인된 green 수소
- 현실성: 학교에 적용하기 쉬운 전략과 조건부 수소의 적용 난이도
